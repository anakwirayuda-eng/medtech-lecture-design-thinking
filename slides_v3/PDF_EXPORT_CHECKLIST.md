# PDF / Export Checklist

Deck utama: `slides_v3/index.html`

## Prinsip
Versi terbaik untuk delivery tetap `HTML live`.  
PDF sebaiknya dipakai sebagai:
- backup jika browser/proyektor bermasalah
- bahan arsip
- bahan berbagi ke panitia atau kolega

Karena itu, cek PDF sebagai `backup format`, bukan pengganti pengalaman live sepenuhnya.

## Sebelum export
- buka [index.html](D:/Dev/Design%20Thinking/slides_v3/index.html) di browser final
- pastikan semua gambar muncul
- pastikan semua mode demo mati:
  - `B` off
  - `X` off
  - `A` off
  - `C` off
  - `P` off
- pastikan deck berada di slide mana saja; print stylesheet akan tetap mengekspor semua slide
- jika Anda pernah klik spoiler saat latihan, tidak masalah; mode print akan tetap aman

## Print to PDF di Chrome / Edge
1. Buka deck di browser.
2. Tekan `Ctrl+P`.
3. Destination: `Save as PDF`.
4. Layout: `Landscape`.
5. Pages: `All`.
6. More settings:
   - `Background graphics`: `ON`
   - `Margins`: `Default` atau `None`
   - `Scale`: `Default` atau `Fit to page`
7. Preview cepat:
   - cek slide gelap masih terbaca
   - cek slide AI dan MRI tidak terpotong
   - cek slide SATUSEHAT masih utuh
8. Save PDF.

## Setelah export
- buka PDF hasilnya
- cek minimal 5 slide ini:
  - hook
  - EMR
  - AI
  - MRI
  - SATUSEHAT
- pastikan jumlah halaman sesuai jumlah slide
- cek apakah teks kecil pada slide case masih terbaca

## Jika hasil PDF jelek
Urutan solusi:
1. aktifkan `P` hanya untuk live, jangan untuk export
2. coba ulang print dengan `Margins: Default`
3. coba browser lain: Edge jika tadi Chrome, atau sebaliknya
4. jika masih jelek, gunakan PDF hanya sebagai handout darurat dan tetap presentasi dari HTML live

## Backup hari-H yang ideal
- `HTML live` sebagai format utama
- `PDF` sebagai backup
- `PRESENTER_ONE_PAGER.md` sebagai pegangan cepat
- `OPENING_RUNTHROUGH_10_MIN.md` untuk warming up sebelum masuk kelas

## Final check 60 detik
- deck terbuka
- PDF tersimpan
- hotkeys masih berfungsi
- browser tidak terlalu banyak tab
- charger dan kabel display siap
